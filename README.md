## アプリケーション名
### questweet
## アプリケーション概要
このアプリケーションでは、質問を投稿しそれをさまざまなユーザーが回答し、解決する事ができます。

## 利用方法
メールアドレス、パスワード、ユーザー名、プロフィールを入力してユーザー登録を行う事ができます。
質問登録ページで、質問のタイトルと詳細を記述することで、質問一覧ページに質問が載ります。
質問回答ページでは、ログインしているユーザーのみ回答をする事ができます。
質問回答ページで、不適切の単語が含まれていた場合は、「不適切な表現が含まれています」というコメントに変換されます。
コメントしたURLをクリックするとそのページに飛ぶ事ができます。
質問したユーザー本人は質問を消す事ができます。
## 目指した課題解決
自身が疑問に思ったことや解決したいことを簡単に発信する事ができるコミュニティの形成の手助けをする事ができます。
## 洗い出した要件
### ユーザー機能
- ユーザーがログインをしていない時は、新規登録、ログインボタンが表示される
- ユーザーがログインをしている場合は、ログアウト、ユーザー情報編集、質問投稿ボタンが表示される
- ログイン状態に関わらず質問一覧は表示される

### 新規登録機能
- メールアドレス、パスワード、ニックネーム、プロフィールを入力する事でユーザー登録ができる
- メールアドレスを入力しなければ登録ができない
- パスワードを入力しなければ登録ができない
- パスワードを入力しても、確認のため２回入力しないとを入力しなければ登録ができない
- ニックネームを入力しなければ登録ができない
- プロフィールを入力しなければ登録ができない

### ログイン機能
- メールアドレスとパスワードを入力し、登録されていればログインする事ができる
- メールアドレスを入力しなければログインできない
- パスワードを入力しなければログインできない
- 登録されていなければ、メールアドレスとパスワードを入力してもログインする事ができない

### ユーザー編集機能
- ニックネーム、プロフィール、メールアドレスを変更して登録する事ができる
- 最初は元の情報が入力されている状態で表示される
- ニックネームを入力していない時は登録できず、編集ページにとどまる
- プロフィールを入力していない時は登録できず、編集ページにとどまる
- メールアドレスを入力していない時は登録できず、編集ページにとどまる
### 質問登録機能
- 質問のタイトルと詳細を入力することで質問を登録する事ができる
- 登録された質問のタイトルのみ質問一覧ページに表示される
- タイトルが入力されていない時質問を登録する事ができず、質問登録ページにとどまる
- 詳細が入力されていない時質問を登録する事ができず、質問登録ページにとどまる

### 質問表示機能
- 質問のタイトルと詳細がページ上部に表示される

### 質問回答機能
- ログイン状態のユーザーのみコメントで返信ができる
- ログイン状態で質問したユーザー本人のみ質問を削除する事ができる
- ログインしていないユーザーには「コメントの投稿には新規登録/ログインが必要です」という文字列が表示される
- 空のコメントは保存されない
- 入力したコメントはコメント一覧に載る
- コメントにURLが含まれていた場合それをクリックすることでページに飛ぶ事ができる
- 不適切な表現があった場合は「不適切な表現が含まれています」というコメントに変わる
- URL以外の文字列はクリックする事ができない
